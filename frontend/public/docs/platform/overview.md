# Platform Overview

Chronos Studio provides a comprehensive platform for building, deploying, and managing AI agents.

## Components

### Dashboard
The web-based interface for managing agents, viewing analytics, and configuring settings.

### Jestha Platform
Custom deployment infrastructure optimized for AI workloads.

### Spark Platform
High-performance compute platform for training and inference.
