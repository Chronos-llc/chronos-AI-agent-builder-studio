# Product Roadmap

Our vision for the future of Chronos Studio. Roadmap items are subject to change based on community feedback.

---

## Q2 2026 — Expansion

### Agent Capabilities
- [ ] Agent-to-agent marketplace — share and monetize blueprints
- [ ] Advanced autonomous scheduling — complex multi-step cron workflows
- [ ] Agent versioning and A/B testing
- [ ] Custom embedding model support for memory

### Voice AI
- [ ] Real-time language translation during calls
- [ ] Multi-party voice calls (conference)
- [ ] Voicemail detection and handling
- [ ] Outbound campaign management dashboard

### Integrations
- [ ] Native CRM integrations (HubSpot, Salesforce)
- [ ] Zapier / Make.com connectors
- [ ] Native database connectors (PostgreSQL, MongoDB)
- [ ] Calendar integration (Google, Outlook)

## Q3 2026 — Enterprise

### Platform
- [ ] Self-hosted deployment option
- [ ] Multi-region data residency
- [ ] Advanced RBAC (role-based access control)
- [ ] Audit logging and compliance dashboard
- [ ] SSO via SAML and OIDC

### Developer Experience
- [ ] Visual agent builder (drag-and-drop)
- [ ] Agent debugging timeline — step-by-step execution replay
- [ ] Performance profiling and optimization suggestions
- [ ] GitHub Actions for CI/CD deployment

## Q4 2026 — Scale

### Jestha Workspace
- [ ] Desktop app (macOS, Windows, Linux)
- [ ] Mobile app (iOS, Android)
- [ ] Team collaboration — shared workspaces
- [ ] Workflow templates library

### Voice AI
- [ ] Custom voice model training
- [ ] Video call support (WebRTC with camera)
- [ ] Real-time agent coaching — whisper mode for human agents
- [ ] Compliance recording with PCI-DSS support

### Ecosystem
- [ ] Plugin marketplace
- [ ] Partner API program
- [ ] Developer certification program
- [ ] Open-source community tools
