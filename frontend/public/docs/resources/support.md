## Troubleshooting

### Common Issues

**API Errors**
- Verify API key is valid
- Check rate limits
- Ensure proper headers

**Agent Not Responding**
- Check agent status
- Review logs
- Verify configuration

**Voice Call Issues**
- Check network quality
- Verify telephony setup
- Review call logs
