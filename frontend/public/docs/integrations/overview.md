# Integrations Overview

Chronos Studio integrates with a wide range of services and tools to extend your agents' capabilities.

## Categories

### API Integrations
Connect agents to external APIs and services.

### Database Integrations
Store and retrieve data from databases.

### MCP (Model Context Protocol)
Standardized integration with AI tools and services.

### Messaging Platforms
Deploy agents on messaging platforms like Slack, Discord, and more.
