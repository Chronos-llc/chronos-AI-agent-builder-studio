## Quick Start

Get your first agent running in under 5 minutes:

```bash
# Install the Chronos CLI
npm install -g @chronos-studio/cli

# Authenticate
chronos login

# Create your first agent
chronos init my-first-agent

# Deploy
chronos deploy
```
